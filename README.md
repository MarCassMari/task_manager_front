# Task Manager Front-End

Este é o front-end do Task Manager - "to do", uma aplicação React que consome os serviços de uma API para gerenciar tarefas.

## Funcionalidades

- Listar tarefas
- Adicionar novas tarefas
- Editar tarefas existentes
- Excluir tarefas
- Marcar tarefas como concluídas

## Pré-requisitos

Certifique-se de ter instalado:

- [Node.js](https://nodejs.org/) (versão 16 ou superior)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

## Instalação

1. Clone este repositório:

   ```bash
   git clone https://github.com/seu-usuario/task-manager-front.git
   ```
